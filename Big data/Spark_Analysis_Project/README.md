This is a folder for a project the implements comprhensive analytics using SPARK 

Project objectives:

* Perform exploratory data analysis and descriptive statistics on Chicago crime incident records.
* Build a classification model to predict whether an incident will lead to an arrest.
* Perform clustering to group similar crime patterns using time, crime category, and police-area features.
* Build a streaming data pipeline with Apache Kafka and Apache Flink to process incident events in near real time.
